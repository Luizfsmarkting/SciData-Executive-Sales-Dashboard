## Project Overview
This project features a high-level executive dashboard designed for a US-based CEO to monitor sales performance and logistics for a major e-commerce operation. Using the **Olist Brazilian E-commerce dataset**, I transformed raw data into a clean, intuitive, and internationalized business tool for the fictional company **SciData**.

## Key Features & Insights
- **Executive KPIs:** Real-time tracking of GMV (Gross Merchandise Value), Total Orders, and AOV (Average Order Value).
- **Time Intelligence:** Month-over-Month (MoM) revenue trends and growth analysis.
- **Supply Chain Analysis:** Delivery performance tracking and product category ranking.
- **International Standards:** Fully localized for the US market (Currency, Date formats, and English business terminology).

## Technical Implementation
- **Data Engineering (ETL):** Conducted full data transformation using **Power Query (M Language)** to handle date inconsistencies and localized formatting.
- **Data Modeling:** Implemented a **Star Schema** architecture to ensure optimal report performance and scalability.
- **DAX Measures:** Developed advanced DAX formulas for dynamic formatting, time intelligence, and strategic KPIs.
- **UI/UX Design:** Applied minimalist design principles, utilizing shadows, corporate color palettes, and visual hierarchy.
